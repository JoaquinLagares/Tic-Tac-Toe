# Tic-Tac-Toe

A simple web-based Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## Features

- Two-player mode with custom player names
- Interactive 3x3 game board
- Displays win, tie, and turn messages
- Restart and start game functionality

## How to Play

1. Enter player names in the input fields.
2. Click **Start Game** to begin.
3. Players take turns clicking on the board to place their mark (X or O).
4. The game announces the winner or a tie.
5. Click **Restart** to play again.

## Project Structure

- [`index.html`](index.html): Main HTML file
- [`styles.css`](styles.css): Styling for the game
- [`main.js`](main.js): Game logic and interactivity
- [`readme.md`](readme.md): Project documentation

## Demo

Open `index.html` in your browser to play.
